🔌 CryptoFlow Widget Test

This page simulates an external website embedding our widget

How it works: The widget is loaded via a single <script> tag and configured using data-* attributes. No React, no npm, just vanilla JavaScript!
Bitcoin - Dark Theme
Ethereum - Light Theme
Solana - With Exchange Link
Cardano - Minimal (no stats)
Polkadot - Dark Theme
Dogecoin - Light Theme

📋 Integration Code

Copy and paste this code into any HTML page:

<!-- Step 1: Add widget container wherever you want it -->
<div
    data-widget="crypto-price"
    data-coin="bitcoin"
    data-theme="dark"
    data-exchange="Binance">
</div>

<!-- Step 2: Add script before </body> (only once per page) -->
<script src="/widget.js"></script>

Configuration Options

Attribute Description Values Default
data-coin Cryptocurrency to display bitcoin, ethereum, solana, etc. bitcoin
data-theme Widget color theme dark, light dark
data-currency Display currency usd, eur, gbp usd
data-exchange Show exchange trade link Binance, Coinbase, etc. none
data-show-volume Show 24h volume true, false true
data-show-market-cap Show market cap true, false true
data-refresh-interval Auto-refresh interval (ms) Any number (0 to disable) 60000
